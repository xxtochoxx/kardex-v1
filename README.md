# Kardex-v1  
Sistema de registro y control de inventarios (Kardex) desarrollado en **Python**.

Este proyecto comienza como un backend en Python y se expandirá en futuras versiones para incluir una interfaz web (HTML/JS) y una base de datos PostgreSQL. La arquitectura está pensada para ser modular, escalable y fácil de mantener.

---

## Tecnologías previstas

- **Python 3.10+**
- **FastAPI** (API REST)
- **Uvicorn** (servidor ASGI)
- **SQLAlchemy** (ORM, opcional)
- **PostgreSQL** (planeado para versiones futuras)
- **Pytest** (testing)
